# pdf reference converter
主要是把pdf引文部分不需要的回车去掉，然后提取里面的信息，生成链接，点击链接就可以进入百度学术里面相应的文献页面。省去复制粘贴的劳动。由于不同的杂志有不同的引文格式，目前只是做了一个杂志的样式，接下来还需要在其他的杂志上面测试才可以。

## 目前支持的杂志
- J Mol Hist

## 使用方式
- 将pdf引文文本复制粘贴到ref.txt
- 利用node.js运行extractDOI.js
- 打开生成的search.html
